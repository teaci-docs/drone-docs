Get build logs

Returns the build logs for a specific job (build step).

GET /api/repos/{owner}/{name}/logs/{number}/{job}

Cancel a Job

Cancel the a build job by number.

DELETE /api/repos/{owner}/{name}/logs/{number}/{job}
